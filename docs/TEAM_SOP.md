# 👥 SOP Team / Studio YouTube

---

## Vai trò trong team

### 1-Person Setup (Solo)
```
Bạn = Researcher + Scriptwriter + Editor + Manager
```
- Dùng AI để hỗ trợ mọi bước
- Target: 1 long-form/ngày + 3-5 shorts/ngày

### 2-Person Setup
```
Person 1: Research + Script + Voice
Person 2: Edit + Thumbnail + Upload
```

### 3-Person Setup
```
Person 1 (Producer): Research + Script
Person 2 (Editor): Edit + Thumbnail
Person 3 (Manager): Upload + SEO + Analytics
```

### Studio Setup (5+ người)
```
Researcher (1-2): Tìm nguồn + Research
Scriptwriter (1-2): Viết script
Voice Artist (1): Voice over
Editor (2-3): Edit video
Thumbnail Designer (1): Thiết kế thumbnail
Channel Manager (1): Upload + SEO + Analytics + Community
```

## KPI & Metrics

### KPI hàng ngày
| Metric          | Target Shorts | Target Long-form |
|-----------------|---------------|-------------------|
| Videos/ngày     | 3-5           | 1                 |
| Thời gian/video | 30 phút       | 2-3 giờ           |

### KPI hàng tuần
| Metric              | Target       |
|----------------------|-------------|
| Tổng views           | 50,000+     |
| Subscribers gained   | 500+        |
| Avg CTR              | > 5%        |
| Avg Retention        | > 40%       |
| Revenue              | Track       |

### KPI hàng tháng
| Metric              | Target       |
|----------------------|-------------|
| Tổng views           | 200,000+    |
| Subscribers total    | Track       |
| Monthly revenue      | Track       |
| Best performing      | Analyze     |
| Worst performing     | Learn       |

## Meeting Schedule

### Daily standup (15 phút)
- Hôm qua làm gì?
- Hôm nay làm gì?
- Có blocker gì?

### Weekly review (30 phút)
- Review KPI tuần
- Top/bottom performing videos
- Adjust strategy

### Monthly planning (1 giờ)
- Review KPI tháng
- Lên content calendar tháng tới
- Đánh giá ngách
- Budget review
